Pseudocode of an everyday task: Performing CPR

  1. Check if there is a qualified professional around
  2. Lay person on their back and open airways
  3. Make sure if they are breathing
  4. If person is not breathing proceed to begin CPR
  5. Start with 30 chest compressions
    • Use 2 hands for adults
    • 1 hand for children
    • 2 fingers for infants
    • Let chest rise in between each compression
  6. Then do two rescue breaths
  7. Repeat process until a Medical Professional arrives on scene

Pseudocode of a computer task: Creating a Hangman Game

  1. Come up with a word at random
  2. Display the length of the word to the user
  3. Correct_guesses is less than the length of the word
  4. Tell the user to guess a letter
  5. If the user guesses a correct letter, then user earns a point for correct_guesses
  6. If the user guesses an incorrect letter, then the user earns a point for incorrect_guesss and draw the next part of hangman.
  7. If the users incorrect_guesses are greater than 8, then the user has lost the game
  8. If the users correct_guesses are equal to the length of the word, then the user has won the game
